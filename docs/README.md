# Documentation

This documentation follows the Diataxis model. Each section serves a
specific need.

## Tutorials (learning-oriented)

- `docs/tutorials/first-notification.md`: run Moodle Mate and send a first
  test notification.

## How-to guides (task-oriented)

- `docs/how-to/setup-dev-environment.md`: set up a local dev environment.
- `docs/how-to/configure-web-ui.md`: enable and access the Web UI locally.
- `docs/how-to/add-custom-provider.md`: add a custom notification provider.

## Reference (information-oriented)

- `docs/reference/configuration.md`: configuration fields and defaults.
- `docs/reference/cli.md`: CLI commands and flags.

## Explanation (understanding-oriented)

- `docs/explanation/architecture.md`: pipeline and module structure.
- `docs/explanation/security.md`: local-only Web UI and session cache trade-offs.
